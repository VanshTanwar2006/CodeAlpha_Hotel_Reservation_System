# Hotel Reservation System 🏨

A desktop-based application developed in Java using the Swing framework. This project simulates a real-world hotel booking experience with user input validation and dynamic billing.

## 🚀 Features
* **Interactive UI:** Built using Java Swing and AWT for a smooth desktop experience.
* **Room Management:** Offers selection between Standard, Deluxe, and Luxury Suites with real-time pricing.
* **Automated Calculations:** Computes the total bill dynamically based on the number of days stayed.
* **Simulated Payment Gateway:** Generates a professional receipt with a random Transaction ID upon successful booking.
* **Robust Exception Handling:** Validates fields to ensure empty strings or negative numbers are not processed.

## 🛠️ Technologies Used
* **Language:** Java Core
* **Framework/Library:** Java Swing, AWT
* **Tools:** VS Code, Git, GitHub
